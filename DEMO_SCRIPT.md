# Demo Script: Multilingual Chat Reply (≤ 3 min)

## Overview
- **Goal**: Show the chat interface responding in **English**, **Hindi**, and **Marathi**.
- **Duration**: Up to 3 minutes.
- **Prerequisites**: App opened, TTS enabled, a citizen profile loaded.

## Script Timeline
| Time | Action | Narration / On‑Screen Text |
|------|--------|-----------------------------|
| 0:00‑0:15 | Launch the app | "Welcome to the Scheme Explorer. Let’s see how the chatbot works in three languages."
| 0:15‑0:30 | Switch language to **English** (dropdown) | "Language set to English."
| 0:30‑0:45 | Type query: *"What is the eligibility criteria?*" | Bot replies in English (text + TTS).
| 0:45‑1:00 | Switch language to **Hindi** | "भाषा हिन्दी में बदल दी गई है।"
| 1:00‑1:15 | Type query in Hindi: *"योग्यता मानदंड क्या है?*" | Bot replies in Hindi (text + TTS).
| 1:15‑1:30 | Switch language to **Marathi** | "भाषा मराठी मध्ये बदलली आहे."
| 1:30‑1:45 | Type query in Marathi: *"पात्रता निकष काय आहेत?"* | Bot replies in Marathi (text + TTS).
| 1:45‑2:15 | Click **Generate PDF** button, generate scheme PDF. | Show PDF preview; it contains the same multilingual content.
| 2:15‑2:45 | Play the PDF preview while the bot reads the summary in the selected language. | Demonstrates seamless transition from chat to PDF.
| 2:45‑3:00 | Wrap‑up | "The chatbot can understand and speak in three languages, and the PDF reflects the same information. Thank you!"

## Key Points to Emphasise
- Language toggle updates UI text **and** TTS voice.
- Responses are consistent across languages.
- PDF generation respects the current language selection.
- Speech synthesis finishes before the next response (no overlap).

## Tips for a Smooth Demo
- Pre‑load a citizen profile to avoid login delays.
- Use text input (mute microphone) to avoid background noise.
- Verify the browser supports SpeechSynthesis (Chrome/Edge).
- Keep the PDF preview pane open beforehand.

*End of script*.
